Decimal Values

If LP or ATK/DEF are halved by some effect, so that the result has decimal values, the calculation is rounded up[1].

Example Question:
If an effect that of like "Blackwing - Gale the Whirlwind"'s effect halves an ATK of "725" to "362.5", how is the decimal place value calculated?
Answer:
If an effect that of like "Blackwing - Gale the Whirlwind"'s effect causes an ATK or DEF to have a decimal place value, that value is rounded to the nearest whole number.
Therefore, if a value of "362.5" occurs due to "Blackwing - Gale the Whirlwind"'s effect, it becomes "363"[2].
